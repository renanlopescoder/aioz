Banco de Dados

  • O "M" do M.E.A.N é de MongoDB que é um banco de dados orientado a documento multi-plataforma que tem alta performance e de fácil escalabilidade, a grande vantagem de usar o MongoDB em uma aplicação JavaScript é que o banco trabalha com coleções de documentos do tipo chave valor uma estrutura muito parecida com o JSON o que reduz a impedancia da nossa aplicação.

    Saiba Mais: MongoDB Wise Guia Rápido

    O M.E.A.N Pack utiliza o mongoose por padrão que é uma api que facilita o uso do MongoDB em aplicações NodeJS. Nosso arquivo de configuração da base de dados está na pasta config/database.js acessando este arquivo temos o seguinte código.

    // Local database configuration
    // Configuração da Base de dados local
    mongoose.connect ('mongodb://localhost/nomeSeuBanco');

    Para configurar deve criar uma base de dados usando terminal do MongoDB e o passando como parâmetro a função do mongoose.connect após esta configuração seu banco de dados tem que estar rodando para que a aplicação funcione corretamente. Acesse o terminal e execute o comando mongod para inicializar seu MongoDB
