Diretivas

  • Diretivas

    As diretivas são tags ou elementos desenvolvidos para encapsular, tornar código semântico ao projeto e reaproveitar código ajudando na velocidade do desenvolvimento, podemos criar nossas próprias tags com todo html e css definidos para este fim.

    A cada interface de evento do JavaScript temos uma diretiva do angular, para um form submit usamos a diretiva ng-submit= “funcao()” e caso necessário executar uma função através de um link (a) podemos usar o ng-click=“funcao()"

    Diretivas Tags

    Podemos criar nossas próprias tags html com css e html definido, assim ao usar estaremos carregando nossa tag desenvolvida, essas tags não existem no mundo HTML o AngularJS irá “ensinar” ao html.

    • Barra de Navegação Lateral (SideNav)
      Instalação será feita via Homebrew para instalar o homebrew usa o seguinte comando no terminal /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent .com/Homebrew/install/master/install )"
      Instalação do MongoDB via Homebrew primeiramente devemos atualizar o homebrew brew update então podemos instalar o MongoDB brew install mongodb
      Em alguns casos precisamos usar o comando como sudo para criação da pasta db no terminal sudo mkdir -p /data/db e mudar a permissão de acesso usando linha de comando sudo chmod 777 /data/db
      Então para iniciar o MongoDB basta usar o comando mongod
